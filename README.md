# 🚀 FinBoard  
**Real-Time Currency Intelligence & Automated Alerts Platform**  
打造為「投資人 / 外匯交易者」設計的專業級匯率監控 + 自動化提醒平台。

---

## 📌 About  
FinBoard 透過標準 API、即時計算與自動提醒，讓你在不用整天盯盤的情況下，也能掌握最佳買賣機會。  
支援 Web Dashboard、Telegram Bot（可選），並可一鍵部署至 Render。

---

# ⭐ Core Features

## 1️⃣ 即時匯率查詢（Real-Time FX Rates）
- 支援 USD/TWD、USD/JPY、JPY/TWD…等常用貨幣  
- 即時回應（FastAPI + aiohttp）  
- 提供 API 給 Telegram bot 查詢  
- 完整支援 JSON API 格式

---

## 2️⃣ 自動化提醒 & 任務排程（Automated Alerts & Scheduler）
- APScheduler 週期任務  
- Render 雲端定時執行  
- 每日固定時間推播提醒  
- 可自訂監控目標價、門檻  
- Email / Telegram（可擴充）

---
